# 🛒 E-Commerce Sales Performance Analysis | SQL Server

## 📌 Project Overview

This project analyzes the **Online Retail** dataset from the **UCI Machine Learning Repository** using **Microsoft SQL Server**. The objective is to clean, transform, and analyze transactional sales data to uncover valuable business insights related to revenue, customer behavior, product performance, and sales trends.

The project demonstrates advanced SQL techniques including **data cleaning, Common Table Expressions (CTEs), Window Functions, aggregate functions, ranking, customer segmentation, and business KPI analysis** to support data-driven decision-making.

---

## 🎯 Business Objectives

The project aims to answer the following business questions:

- What is the total revenue generated by the business?
- How does monthly revenue change over time?
- Which products generate the highest sales?
- Who are the highest-value customers?
- Which countries contribute the most revenue?
- How frequently do customers make repeat purchases?
- What purchasing patterns can improve business performance?

## 📂 Dataset

**Dataset Name:** Online Retail

**Source:** UCI Machine Learning Repository

🔗 **Official Dataset:**  
https://archive.ics.uci.edu/dataset/352/online+retail

### Dataset Description

The dataset contains transactional sales records from a UK-based online retail company between **December 2010 and December 2011**. It includes **541,909 transactions** from customers across multiple countries and is widely used for sales analysis, customer segmentation, and business intelligence projects.

### Dataset Features

| Column | Description |
|---------|-------------|
| InvoiceNo | Unique invoice number (Invoices beginning with **'C'** indicate cancelled transactions.) |
| StockCode | Unique product identifier |
| Description | Product description |
| Quantity | Number of units purchased |
| InvoiceDate | Date and time of the transaction |
| UnitPrice | Product price per unit (GBP) |
| CustomerID | Unique customer identifier |
| Country | Customer's country |

**Dataset Size:** 541,909 Transactions

**Project Domain:** E-Commerce Sales Analytics
