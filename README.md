# 🛒 E-Commerce Sales Performance Analysis | SQL Server

## 📌 Project Overview

This project analyzes transactional sales data from an online retail company using **Microsoft SQL Server**. The objective is to transform raw sales data into meaningful business insights by applying advanced SQL techniques, enabling data-driven decision-making across sales, customer behavior, and product performance.

The project demonstrates practical SQL skills commonly used by SQL Developers and Data Analysts, including data cleaning, aggregation, window functions, customer segmentation, and business performance analysis.

---

## 🎯 Business Objectives

The project addresses several key business questions, including:

- What is the total revenue generated?
- How do monthly revenue trends change over time?
- Which products generate the highest sales?
- Who are the most valuable customers?
- Which countries contribute the most revenue?
- How frequently do customers make repeat purchases?
- What purchasing patterns can support business growth?
